Is my host fast yet?

A web transparency study using CrUX and HTTP Archive

How it works (briefly)

Hosts are detected by their HTTP header signatures. For example, a response header of x-powered-by: HubSpot for a website's HTML page indicates that it is hosted by HubSpot. The HTTP Archive dataset includes metadata for over 5 million web pages and is updated monthly. We use query this dataset to detect whether the response headers include the signatures for known hosts.

Once we know the websites for each host, we intersect those with the Chrome UX Report dataset. This gives us real user experience data about each website, including TTFB.
