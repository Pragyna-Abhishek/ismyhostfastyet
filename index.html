How it works

Hosts are detected by their HTTP header signatures. For example, a response header of x-powered-by: HubSpot for a website's HTML page indicates that it is hosted by HubSpot. The HTTP Archive dataset includes metadata for over 5 million web pages and is updated monthly. We use query this dataset toÂ detect whether the response headers include the signatures for known hosts.

Once we know the websites for each host, we intersect those with the Chrome UX Report dataset. This gives us real user experience data about the performance each website, including TTFB. These values are rated as fast, average, or slow according to these rules:

Fast
Less than 200 ms
Average
Greater than or equal to 200 ms and less than 1000 ms
Slow
Greater than or equal to 1000 ms
